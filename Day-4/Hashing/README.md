🔐 What Is Hashing?
Hashing is a technique that converts data into a fixed-size value, called a hash.
In DSA, we use hashing to achieve:

🔎 Fast lookups (O(1))

🚫 Duplicate detection

📊 Frequency counting

🧠 Quick mapping between keys and values

In Python, hashing is implemented under the hood in:

   set

   dict
